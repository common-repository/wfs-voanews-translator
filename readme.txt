This plugin helps users to easily insert a translator tool in each post content

== Description ==

Features: 

<ul>
<li>Explaining english word by english phase when user clicks on a word</li>
<li>Supporting responsive display for position the mouse clicking point</li>
</ul>

Admin panel:

<ul>
<li>Plugin could be active or disable</li>
<li>Tooltip background color could be changed with 11 colors</li>
<li>Supporting catching the post content by content tag css class if user uses a special wordpress theme with different css style</li>
</ul>

Visis us: <a href="http://webfacescript.com/category/product-list/product-list-wordpress-plugins/">webfacescript.com</a>

== Installation ==

1. Upload the entire wfs_voanews_translator folder to the /wp-content/plugins/ directory or Add New Plugin by upload the wfs_voanews_translator.zip
2. Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'WFS VOANews Translator' menu in your WordPress admin panel.
